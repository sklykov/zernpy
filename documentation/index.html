Installation instructions

The straightforward way to install the library - to use the following command for the pip package installer: pip install zernpy
The requirements for installation: Python version >= 3.8, libraries numpy and matplotlib. Check the webpage of this library on PyPi package index.

Import and class initialization

The basic import command: from zernpy import ZernPol
The initialization of the ZernPol class instance: zp = ZernPol(m = -2, n = 2). This instance gives an access to the calculation methods related to the Zernike polynomials properties.
Other acceptable initialization parameters names: osa_index / noll_index / fringe_index or for shortness osa / noll / fringe, for example:
zp = ZernPol(osa_index = 3) or zp = ZernPol(osa = 3)
